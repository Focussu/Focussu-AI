## How to run

### 처음 실행 또는 코드 변경 시

> docker compose up --build

### 변경 사항이 없을 때 빠른 실행

> docker compose up

## Test

> localhost:8000

## Develope AI Code

- Install Docker Desktop.
- Install Devcontainer in VSCode
- 좌측하단 ><마크 클릭해서 Reopen in Container 선택
- src, data 디렉토리는 컨테이너 내부와 공유되므로 컨테이너 환경에서 개발
